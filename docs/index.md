# Sistema de Gestión de Concesionario

Documentación oficial del **Sistema de Gestión de Carros Usados**. Este proyecto es una aplicación de terminal (CLI) construida en Python diseñada para administrar el inventario, ventas y precios de un lote de vehículos.


El objetivo principal es aplicar los principios de **Arquitectura Limpia** separando las responsabilidades en capas (Modelos, Servicios, Almacenamiento e Interfaz) garantizando un código mantenible y escalable.

## Características Principales
* **CRUD Completo:** Agrega, lee, actualiza y vende vehículos.
* **Persistencia Local:** Los datos se guardan de forma segura en un archivo JSON.
* **Validación Robusta:** Reglas de negocio estrictas para evitar IDs duplicados o precios negativos.
